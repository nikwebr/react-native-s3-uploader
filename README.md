# react-native-s3-uploader

chunked background uploader to s3 compatible endpoints

## Installation

```sh
npm install react-native-s3-uploader
```

## Usage


```js
import { multiply } from 'react-native-s3-uploader';

// ...

const result = multiply(3, 7);
```
